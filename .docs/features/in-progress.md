## Guidelines

When moving a feature to in-progress:
1. Create a detailed feature spec file if not already exists
2. Update the feature YAML frontmatter
3. Move from backlog.md to this file
4. Assign to developer/team
5. Set estimated completion date

When completing a feature:
1. Update feature spec with actual completion date
2. Move to completed.md
3. Update changelog.md
4. Close related GitHub issues
